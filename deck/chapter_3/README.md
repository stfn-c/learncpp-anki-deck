## Key Topics Covered

### Error Types
- Syntax errors vs semantic errors
- Compile-time vs runtime errors
- Common error patterns

### Debugging Process
- Five-step debugging methodology
- Finding vs fixing bugs
- Incremental development

### Debugging Techniques
- Code inspection
- Print statements (std::cout, std::cerr)
- Commenting out code
- Binary search debugging

### Debugger Usage
- **Stepping**: Step into, step over, step out
- **Breakpoints**: Setting, using, conditional breakpoints
- **Variable inspection**: Watch windows, locals window
- **Call stack**: Understanding execution flow, stack frames

### Prevention
- Compiler warnings (-Wall, -Werror)
- Static analyzers (cppcheck, clang-tidy)
- Unit testing
- Defensive programming
